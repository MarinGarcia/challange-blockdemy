# challange-blockdemy
Rick and Morty GraphQL API

* Technologies Version
node v16.15.1
npm 8.11.0
mongodb 6.0

* System dependencies
```terminal
npm install
```

* Configuration
Create environment variables
DB_USER=
DB_PASSWORD=
DB_NAME=
PORT=

* Database initialization
```terminal
npm run seed
```

* How to run
```terminal
npm run dev
```

* Route
http://localhost:3000/graphql
